## 1. 安装

```bash
npx create-strapi-app@latest my-project
# or
yarn create strapi-app my-project --quickstart
```

## 2. 设置为中文

新建 src/app.js，并粘贴以下内容：

```js showLineNumbers
const config = {
    locales: ['zh'],
};

const bootstrap = (app) => {
    console.log(app);
};

export default {
    config,
    bootstrap,
};
```

重启后进入网站首页，点击左下角头像→个人资料→体验→界面语言，调整为中文即可。

![Untitled](/static/images/blog/202304/Strapi_introduction/Untitled.webp)

## 3. 新建表

默认已经创建了 **User** 表，点击 **创建一个新的 Content Type** 来创建表新表：

![Untitled](/static/images/blog/202304/Strapi_introduction/Untitled%201.webp)

## 4. 创建关系

在一个表中新建一个字段，类型选择**引用**，可以选择关系的类型，包括1v1、1v多、多v1等等。下面是 User 多对一 Article ：

![Untitled](/static/images/blog/202304/Strapi_introduction/Untitled%202.webp)

## 5. 放开 权限

设置 → 角色 → public

开放文章查询权限：

![Untitled](/static/images/blog/202304/Strapi_introduction/Untitled%203.webp)

开放文章作者查询权限，否则连表查询的时候查不到作者信息：

![Untitled](/static/images/blog/202304/Strapi_introduction/Untitled%204.webp)
