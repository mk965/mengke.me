# Mermaid 图表功能测试

这篇文章用于测试 Mermaid 图表功能是否正常工作。

## 简单流程图

```mermaid
graph TD;
    A-->B;
    A-->C;
    B-->D;
    C-->D;
```

## 详细序列图

```mermaid
sequenceDiagram
    participant Host
    participant Client
    participant Server

    Host->>+Client: 初始化 client
    Client->>+Server: 使用 Client 的能力初始化一个会话
    Server-->>Client: 返回 Server 支持的能力
```

## 复杂流程图

```mermaid
graph TB
    A[开始] --> B{需要登录?}
    B -->|是| C[跳转到登录页]
    B -->|否| D[显示主页内容]
    C --> E[验证用户凭据]
    E --> F{验证成功?}
    F -->|是| D
    F -->|否| G[显示错误信息]
    G --> C
    D --> H[加载用户数据]
    H --> I[渲染页面组件]
    I --> J[结束]
```

## 类图示例

classDiagram
    class User {

## 时序图

```mermaid
sequenceDiagram
    participant 用户
    participant 浏览器
    participant CDN
    participant API服务器
    participant 数据库

    用户->>浏览器: 访问页面
    浏览器->>CDN: 请求静态资源
    CDN-->>浏览器: 返回CSS/JS
    浏览器->>API服务器: 请求数据
    API服务器->>数据库: 查询数据
    数据库-->>API服务器: 返回数据
    API服务器-->>浏览器: JSON响应
    浏览器-->>用户: 渲染完成页面
```
